In Angular we can also create custom pipes for specific cases of ours. This can be done by creating a pipe class. We give it the decorator @Pipe and after that we define the class which implemets PipeTransform
After that we must specify the conditions in our transform method which is provided by PipeTransform and in order for the pipe to work we must use it. There we specify our conditions. We specify if the function will do additional formatting by arguments passed and apply that in the body of the function.
In our case we've created a pipe for shortening log text to a specific length passed by the user and apply ... in the end.
