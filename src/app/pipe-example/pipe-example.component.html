Pipe Example

{{ detailedMovie.title | uppercase }}

Year: {{ detailedMovie.year}}
Iron Man actor: {{ detailedMovie.mainActor }}
Release Date: {{ detailedMovie.releaseDate | date: "mediumDate" }}
Rating: {{ detailedMovie.rating | number }}
Price: {{ detailedMovie.price | currency: "BGN":true:'3.2-2' }}
Using pipes in Angular saves a lot of time and effort. It has very easy to use and get familiar with syntax. The pipes can be.
  • uppercase
  • lowercase
  • date
  • currency
  • custom pipes
For example the title above is a string and it uses the uppercase pipe to make all the letter uppercase. The syntax for doing it is: title | uppercase.
We can also provide the pipe with parameters to it. For example we added parameters to the currency and to the date to format it even more. The parameters can be passed after adding a colon. Example: data: "mediumDate" To see what types of parameters can be passed we should read the documentation. And for the currency we format it to currency: "BGN":true:3.2-2 which formats the currency to bulgarian leva and formats it also to the second digit after the dot.
